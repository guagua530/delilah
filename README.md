## 部署到 GitHub Pages

### 步骤 1: 创建 GitHub 仓库

1. 登录 GitHub
2. 点击右上角的 "+" 号，选择 "New repository"
3. 输入仓库名称（例如：romantic-page）
4. 选择 "Public"
5. 点击 "Create repository"

### 步骤 2: 上传文件

在本地项目目录中执行以下命令：

```bash
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/你的用户名/仓库名.git
git push -u origin main
```

### 步骤 3: 启用 GitHub Pages

1. 进入你的 GitHub 仓库
2. 点击 "Settings"（设置）
3. 在左侧菜单中找到 "Pages"
4. 在 "Source" 下拉菜单中选择 "main" 分支
5. 点击 "Save"
6. 等待几分钟后，你的网页就会发布在：`https://你的用户名.github.io/仓库名/`
